微店
微店由北京口袋时尚科技有限公司开发，是一家致力于帮助有梦想的人更容易创业的基于社交关系的电商平台，也致力于为微消费者提供有用，有趣，有态度的购物平台。
公司
公司成立于2011年5月19日，现有员工近1000人，微店，2014年1月1日上线，至今已吸引7200万家商家入驻，遍布全球211国家，是全球增长速度较快的电商公司。
